NAME

    ivsetup - begin running the diskTerminal.dynlib file

SYNOPSIS

    ivsetup [args]

DESCRIPTION

    The SCDisks project is a Lua Disks project (part of LuaDisks project)

    SCDisks is an upgraded Disk Manager Forked from the Craptitude Project.

    main forked features
        * disk management
        * complex makefile

    extra features
        * dedicated terminal
        * organization

    Craptitude is the root of this project.
    This project is simply an upgrade for Craptitude, with QOL features added.
