DISKUTILITY(1) - Manual Page

NAME | SYNOPSIS | DESCRIPTION

NAME

    diskutility - A Utility for running .dsi files.

SYNOPSIS

    diskutility [-e ENTRY | --entry=ENTRY] [-A arg] [-v] [FILE]

DESCRIPTION

    It works by initializing the disk into Lua 5.4's memory file stack and then
    runs the file using either two methods, ANY, Or MAIN.

    Main is a predefined function which is based off of the Linux Kernel's C main function.

OPTIONS

    -e | --entry=N,
        Runs the entry "N" from the file, "FILE"
