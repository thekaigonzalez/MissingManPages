NAME

    LKernel - A Lua 5.4.3 Process which runs files using .dsi, .imrc, and .rc file extensions in Lua bytecode format.

SYNOPSIS

    [LKernel]
        [base]
            [functions & files]
            [kernel syscalls]
            [require libraries]
        [lua bytecode]
        [text utf-8 w/ bytes]

DESCRIPTION

    An LKernel is a .dsi file which runs Lua code using luaL_dofile(X, XX); Or the Lua dofile(filehandler); function.

    lua-std-5.4.3 compatibility
        The Compatibility with Lua 5.4 Standard library is seamless, you can view examples in the EXAMPLES
        Header.
