NAME

    process-tables - The process's data access table in Lua 5.*

SYNOPSIS

 
     local tbl = {}
 

     return tbl

DESCRIPTION

    The Process tables is the simplest part of building a LKernel for your applications.
    Process tables are returned tables from the .dsi file. Used in the "MAIN" process, etc...

    The Use of process tables are to give the compiler a direction to go in to run the program.
    Since all programs are ran in stack, the process goes like this:

    lua-stack tables
        [get program]
            [get tbl]
                [run function from tbl]
    Stack tables format are compiled like this.
