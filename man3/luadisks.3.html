NAME

    luadisks - A Lua Project which replicates Kernel Partition Management.

SYNOPSIS

    partitions
    data handling
    terminal (logically)

DESCRIPTION

    The LuaDisks project aims to create certain build systems using Lua binary bytecode.

    the first project
        The first EVER LuaDisks project is called KBOX, which was
        originally supposed to be a MS-DOS Terminal, but after a bit of shenanigans, the project was abandoned,
        and replaced with the Craptitude project, Then thereafter; followed by the SCDisksproject, themost recent one.
