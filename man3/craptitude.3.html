NAME

    craptitude - a Lua Disk manager based off of FreeKSD (LuaDisks)

DESCRIPTION

    Craptitude is a simplistic project containing FreeKSD code and also creates and manages disks
    using the crub2 disk manager, from KBOX.

    Crub2 is based off of FreeKSD and uses libreadline for read handling.
